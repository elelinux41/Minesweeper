# Minesweeper

A Minesweeper game with various board shapes (triangular, quadratic, hexagonal) and customizable difficulty levels

## Requirements
- Webserver
- **PHP 8.4** or higher

## Features
- Multiple board shapes: Triangular, quadratic, and hexagonal grids
- Adjustable board size
- Adjustable bomb density
- screen refresh via AJAX

## Rules
**Objective:** Uncover all fields that do not contain a bomb.
- Tapping or clicking reveals a field.
- The number of a field indicates how many of the fields adjacent to it via edges or corners contain a bomb.
- By long-pressing or left-clicking, a field can be marked with a flag to indicate that it contains a bomb.

## Languages
- English
- German
- Italian
